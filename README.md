# Predicting Disease Spread
This project aims to predict the total_cases regarding Dengue fever disease in two cities, San Juan and Iquitos. Dengue fever is a mosquito-borne disease that occurs in tropical and sub-tropical parts of the world. In this project the test data for each city spanning 5 and 3 years respectively. 

## Dependensies
* Numpy
* Pandas
* Matplotlib
* Seaborn
* Scikit Learn

## Steps
* Import dependencies
* Load Training Features Data and Turn it into a pandas data frame and Visualize
* Build the Model
* Fit, Train and Test Model
